Automatically adds a Buzrr.com button to each post/page on your WordPress blog.

== Description ==

This plugin automatically adds the JavaScript code for a [Buzrr.com](http://buzrr.com/) button to each post/page on your WordPress blog.

The latest documentation can be found at the [WPBuzrr](http://ezbizniz.com/wp/plugins/wpbuzrr) page.

== Installation ==

1. Upload the folder 'buzrrcom-button-plugin' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin from 'Settings/Buzrr Options'

== Frequently Asked Questions ==

= How do I change Buzrr options? =

Go to "Settings"->"Buzrr Options".

= Where is the manual? =

The usage of WPBuzrr is quite self-explanatory, but you can find the latest documentation at the [WPBuzrr](http://ezbizniz.com/wp/plugins/wpbuzrr) page.
